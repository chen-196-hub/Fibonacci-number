
任意の数のフィボナッチ数を求めるメソッド fib を実装せよ

# 実行例
```
irb(main):007:0> fib 1
=> 1
irb(main):005:0> fib 10
=> 55
```

# tip

build
```
docker compose build
```

up
```
docker compose up
```

run test
```
docker compose run app bundle exec rspec fibo_spec.rb
```

down (if you finish)
```
docker compose down
```
